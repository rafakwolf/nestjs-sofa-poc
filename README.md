## Description

POC -- Exposing a graphql schema as rest api

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Accessing the graphql playground

http://localhost:3000/graphql

## Accessing the rest api swagger documentation

http://localhost:3000/api/docs
